> Set up a modern flask web server by running one command.

<Br/>

## Installing / Getting started

```shell
pip install build-flask-app
```

## Usage

```shell
build-flask-app
```

## Running The Application

* For flask development server.

```shell
bash bootDev.sh dev
```

<br>

* For gunicorn server.

```shell
bash bootDev.sh prod
```
